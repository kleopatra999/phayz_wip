New features for pacman-g2

Two new features were recently added to pacman-g2: and new operation and a new Query option.

pacman-g2 -P

The new operation uses the parameter "-P" or "--ps" and lists all processes which are using deleted files. From this information you can confirm if any services are affected and if so, restart them.

pacman-g2 -Qf

The new Query option "-Qf" checks the specified package or all packages to confirm if all the files owned by that package are present. This option is very useful if you believe a package might be incomplete; perhaps you think a library file might be missing. Here's sample output you would see if a file was missing:

pacman-g2 /usr/share/doc/pacman-g2-3.8.2/TODO	No such file or directory.

For full details of all pacman-g2's operations and options, refer to its man page.

Minitube

Watching YouTube videos in a web browser has its problems, since it requires a Flash plugin. Minitube aims to make this easier because it allows you to watch them in a GUI application instead. Minitube's user interface is quite simple, with a search input field that allows you to search videos or channels by keywords. Once you have found what you're looking for you can play the video immediately or add it to a playlist. Often people publish to YouTube educational videos or "screencasts" and Minitube's playlist feature is very useful because you can set up the videos to play in sequence.

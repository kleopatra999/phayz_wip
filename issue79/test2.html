Tips and Tricks

Skeleton files

When a new user is created on Linux, the contents of the 

/etc/skel
directory are copied to the users' home page. This allows the computer's administrator to specify configuration settings and/or have folders already available when the user first logs on. For anyone who is the administrator in their home ( :D ) you might use this option, for example, to encourage everyone to store files by type in specific folders: Videos, Photos, Downloads for example. To do this you create directories of the required names in /etc/skel (as user 'root'). Note that this applies only to newly created users, not existing users.

System information script

Some time ago, Frugalware developer Devil505 and Frugalware user Aptano modified a Perl script which presents information about the GUI environment on the console. Focus on Packages

Quod Libet

I'm sorry to feature another audio player after featuring Pogo in the most recent newsletter but I couldn't help myself. :) Quod Libet is a full-featured audio player developed in Python and uses the GStreamer framework. For anyone who is looking for an alternative to players such as Banshee, Rhythmbox or Amarok, Quod Libet could be just what you're looking for.

Its features include:

  • Album art;
  • Track ratings;
  • Internet radio;
  • Numerous plugins including Scrobbling support, Tray Icon, Alarm Clock and Lullaby.

Quod Libet makes it very easy to play your music in the way you want. It offers many different ways to browse or search your music and assemble a playlist. If you need to correct your music's tags you can do this easily on either individual files or several files at once. Tag editing even includes some common built-in functions including replacing underscores with spaces, removing characters which are incompatible with Windows and removing non-ASCII characters.

One quite interesting feature is its Tray Icon plugin. Of course most audio players provide system tray support so that you can continue listening to a playlist without the player's window cluttering your desktop. Quod Libet's tray icon is better than most though because, with the mouse cursor over the icon, scrolling the mouse's wheel will change volume and holding down the Shift key and scrolling the wheel will change track. As with other music players, you can also right-click on the icon and get more functions.

Even if you're currently using a music player, try Quod Libet because it offers a great range of features while still being very easy to use.
